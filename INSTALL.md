## Installation

* We assume this directory is installed in your home, i.e. the paths should be `~/SmarTrim-Artifact`.

* Run

```bash
$ chmod +x wrapper/*.sh
$ wrapper/build.sh
```

* *Estimated build time: 26 minutes*

* This command will install `uv` (A Python package manager) and 13 docker images in your machine.

### Basic Testing

To validate that SmarTrim is installed correctly, you can run a command from the 'A Running Example' section in the [USAGE.md](./USAGE.md) file.
